# Habit-Tracker
This is a simple Job Search web application built using HTML, CSS, and JavaScript. The HTML file creates the basic structure of the website, including a header section with a title and tagline, a search input box, and a grid area to display job listings. The CSS file (jobs.css) is used to design a clean and organized layout with styled job cards, proper spacing, and responsive design for better user experience. The JavaScript file (jobs.js) adds functionality by dynamically displaying job listings and filtering them in real time as the user types in the search box. This project demonstrates basic front-end concepts such as DOM manipulation, event handling, dynamic content rendering, and simple search functionality in a practical way.
